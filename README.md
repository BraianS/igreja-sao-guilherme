# Igreja São Guilherme

Site da Igreja São Guilherme desenvolvido para divulgação da comunidade, informações sobre missas, eventos, contatos e atividades pastorais.

## ✨ Tecnologias

Este projeto foi desenvolvido utilizando:

* Next.js 16
* React 19
* TypeScript
* Tailwind CSS 4
* shadcn/ui
* Framer Motion
* Lucide React
* Radix UI
* date-fns

## 🚀 Como executar o projeto

### Pré-requisitos

* Node.js 20 ou superior
* npm, yarn, pnpm ou bun

### Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/igreja-sao-guilherme.git
```

Acesse a pasta do projeto:

```bash
cd igreja-sao-guilherme
```

Instale as dependências:

```bash
npm install
```

### Executando em ambiente de desenvolvimento

```bash
npm run dev
```

Abra o navegador em:

```text
http://localhost:3000
```

## 📦 Build para produção

Gerar a versão de produção:

```bash
npm run build
```

Executar a aplicação em produção:

```bash
npm run start
```

## 🔍 Lint

Executar a análise de código:

```bash
npm run lint
```

## 🎯 Objetivo

O objetivo deste projeto é oferecer uma presença digital moderna para a Igreja São Guilherme, permitindo que membros e visitantes encontrem facilmente informações sobre:

* Horários de celebrações
* Eventos e avisos
* Informações da comunidade
* Formas de contato
* Localização da igreja
